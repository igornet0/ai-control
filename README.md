# AI Control System

A comprehensive code execution system with real-time logging and Docker deployment.

## Quick Start

```bash
# Start with Docker
docker-compose up --build

# Access services
# Frontend: http://localhost:3000
# Backend: http://localhost:8000
# RabbitMQ: http://localhost:15672
```
